# Ticket Management React

A modern, responsive ticket management system built with React that integrates directly with Zendesk for customer support ticketing.

## Features
- User Authentication: Secure login and signup flows using Clerk.
- Dashboard: View and manage your Zendesk support tickets.
- Ticket Creation: Easily create new support tickets with priorities and types.
- Detailed View: View ticket details, history, and add comments.
- Search and Filtering: Real-time search by keyword or subject and status-based filtering.
- Pagination: Easily navigate through large lists of tickets.

## Technologies Used
- Frontend Framework: React (via Vite)
- Routing: React Router DOM
- State Management: Redux Toolkit (with RTK Async Thunks)
- Styling: Tailwind CSS
- Authentication: Clerk React SDK
- API Client: Axios
- Icons: React Icons
- External API: Zendesk Support API
